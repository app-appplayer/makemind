AutoStudio - makemind 구현체

makemind 철학을 실현하는 완전한 구현체 생태계

AutoStudio는 makemind의 "모든 제어를 자연어로, 모든 앱은 서버로"라는 철학을 실현하는
AppPlayer, MCP Servers, 개발 도구들을 포함한 전체 구현체 생태계입니다.

AppPlayer

Universal MCP Client

모든 MCP 서버를 앱처럼 실행하고 동적 UI를 렌더링하는 유니버설 플레이어

  • MCP 서버 자동 발견 및 연결
  • makemind UI DSL 실시간 렌더링
  • 다중 전송 프로토콜 지원 (STDIO, SSE, HTTP)
  • 크로스 플랫폼 (iOS, Android, Desktop)
GitHub

MCP Servers

Apps as Servers

UI와 도구, 리소스를 포함한 서버 = 앱 구조

  • Server is a complete app
  • UI 정의 + 도구 + 프롬프트 포함
  • LLM이 이해하고 제어 가능
  • 실시간 업데이트 지원

Development Tools

  • 실시간 미리보기 및 테스트
  • 원클릭 배포 및 공유

New possibilities enabled by the MCP ecosystem

AI Autonomous App Development

Smart Home Integration
