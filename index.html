Core Packages

Perfect harmony of specialized packages for each layer

mcp_client

v1.0.0

Client implementation for connecting Flutter/Dart applications to MCP servers

  • MCP 2025-03-26 standard support
  • Built-in OAuth 2.1 authentication
  • Multiple transport types (STDIO, SSE, HTTP)
  • Real-time resource subscriptions
Production Ready

mcp_server

v1.0.2

Server implementation for exposing data and functionality from Flutter apps to AI

  • Expose Resources, Tools, Prompts
  • Session management and event streams
  • Performance monitoring and metrics
  • Multiple protocol version support
Production Ready

mcp_llm

v1.0.1

Advanced package for integration with various LLM providers

  • Claude, OpenAI, Together AI support
  • 40-60% performance improvement with batch processing
  • Multi-client load balancing
  • RAG capabilities and vector search
Production Ready

flutter_mcp

v1.0.3

Complete agent system integrating MCP components

  • Background service execution
  • System tray support (desktop)
  • Secure storage and encryption
  • Platform-specific native implementation
Production Ready

flutter_mcp_ui_runtime

v0.1.0

Runtime engine for rendering dynamic UI from JSON definitions

  • 65+ Flutter widget support
  • Reactive state management
  • Expression binding
  • MCP protocol ready
Beta

mcp_ui_builder

v1.0.0

Visual builder for creating MCP UI with drag and drop

  • Real-time preview
  • Property editor
  • JSON import/export
  • Template library

Getting Started

Create your first MCP application in 5 minutes

# Install Flutter MCP
$ flutter pub add flutter_mcp

# Create a simple MCP server
$ dart create -t console my_mcp_server
$ cd my_mcp_server
$ dart pub add mcp_server

# Run UI Builder
$ git clone https://github.com/yourusername/mcp_ui_builder
$ cd mcp_ui_builder
$ flutter run
